KOMODO CAFE

1. See full menu
=> select this option to see all available menu items
=> from this screen you can see the ingredients in an item by simply typing it's name
=> to exit, type F

2. Add new menu item
=> use this option to add a new meal to the menu
=> follow the prompts to input meal information
=> while entering ingredients, make sure to press enter after each ingredient, and F when you are done

3. Remove menu item
=> use this option to remove an item from the menu
=> simply type the name of the item you wish to remove and press enter
=> to escape without removing an item, simply hit enter

4. Exit
