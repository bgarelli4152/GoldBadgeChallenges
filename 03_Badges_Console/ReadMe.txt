KOMODO SECURITY

1. Add a badge
=> use this option to create a new badge
=> first, enter a numerical badge id
=> next, add a door to access
=> after adding, you may add another door by typing Y, or quit by typing N

2. Edit a badge
=> use this option to add or remove door access
=> enter the id of the badge you wish to access, then type A to add a door, or R to remove a door

3. List all badges
=> use this option to view all active badges

4. Exit
