KOMODO INSURANCE

1. See all claims
=> use this option to see a list of all open claims

2. Handle next claim
=> use this option to handle the oldest open claim
=> select Y to close the claim, or N to pass

3. Create a claim
=> use this option to create a new claim
=> enter valid claim type first (car, home, theft)
=> follow prompts to complete claim entry

4. Delete a claim
=> use this option to delete an open claim
=> please note that closed claims cannot be deleted
=> after reviewing the list of open claims, type the id of the claim you wish to delete

5. See closed claims
=> use this option to review a list of all closed claims
